Declarations

Declaration Primitives

  • var

    Declares a variable, optionally initializing it to a value. If not initialized, it will have the value of undefined.

  • let

    Declares a block-scoped, local variable, optionally initializing it to a value. If not initialized, it will have the value of undefined.

  • const

    Declares a block-scoped, read-only named constant.

Identifier Naming Rules

A JavaScript identifier must start with a letter, underscore (_) or dollar sign ($).
