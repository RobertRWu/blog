static_cast

Description

Converts between types using a combination of implicit and user-defined conversions.

Return

Return a value of the new type.

Properties

  1. Can’t convert const and volatile.
  2. Can be used to convert between base class pointer and subclass pointer.
    • Upcast, converting subclass pointer to base class pointer is safe.
    • Downcast, converting base class pointer to subclass pointer is not safe (No runtime checks).

Examples

c++

const_cast

Description

Can be only used to cast away (remove) constness or volatility of pointers or reference.

Properties

  • Pointer to a const type can be converted to a non-const type, and still point to the same object.
  • Reference is the same.

Examples

c++

reinterpret_cast

Description

Used to convert between different types of pointers (reference) or bewteen integral types and pointers (reference).

Properties

  • Can be used to convert a pointer (reference) to another type of pointer (reference).
  • Can be used to convert a integral type to a pointer or reference which is small than or equal to the integral type and vice versa.

dynamic_const

Description

Safely converts pointers and references to classes up, down, and sideways along the inheritance hierarchy.

Return

  • Succeeded

    Return a pointer to the class.

  • Failed

    Return NULL.

Properties

  • Can only be used with pointers or reference.
  • dynamic_cast is dealing while running.
  • Can’t be used to convert between built-in type.
  • There must be virtual functions in the class.
