Expressions

Operators

Unary Operators

  • ++, --

    Increment and decrement.

  • +

    Converts its operand to Number type.

  • -

    Converts its operand to Number type and then negates it.

  • typeof(operand)

    Return the type of a given object in the form of string.

  • void(expression)

    Discards an expressionâ€™s return value.

Logical Operators

  • &&

    Logical AND

  • ||

    Logical OR

  • !

    Logical NOT

Equality Operators

  • ==

    Equality operator.

  • !=

    Inequality operator.

  • ===

    Identity operator.

  • !==

    Nonidentity operator.

Relation Operators

  • in

    Determines whether an object has a given property.

  • <

  • >

  • <=

  • >=

Arithmetic operators

  • +

  • -

  • /

  • *

  • %

  • **

    Exponentiation operator.

Bitwise Operator

  • &

    AND

  • |

    OR

  • ^

    XOR

  • ~

    NOT

  • <<

    Left shift

  • >>

    Arithmetic right shift

  • >>>

    Logical right shift

Ternary Operator

  • condition ? ifTrue : ifFalse

Comma Operator

  • ,

    Allows multiple expressions to be evaluated in a single statement and returns the result of the last expression.
