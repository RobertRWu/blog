HTML List

Introduction

There are three different types of HTML lists:

  1. Ordered List (ol)
  2. Unordered List (ul)
  3. Description List (dl)

HTML Ordered List

The ordered list starts with <ol> tag and the list items start with <li> tag:

html

Here is how it looks:

  1. Amy
  2. Bob
  3. Robert

HTML Unordered List

The Unordered list starts with <ul> tag and list items start with the <li> tag.

html

Here is how it looks:

  • Amy

  • Bob

  • Robert

HTML Description List

The definition list is very appropriate when you want to present glossary or list of terms.

The HTML definition list contains following three tags:

  1. <dl> tag defines the start of a list
  2. <dt> tag defines a term
  3. <dd> tag defines the term’s description
html

Amy

​ -My friend.

Bob

​ -My friend too.

Robert

​ -Me.

HTML Nested List

A list within a list. It’s easy to know how to do it, just change items in the outer list into lists.
