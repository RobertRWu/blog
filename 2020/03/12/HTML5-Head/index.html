HTML Head

Introduction

The HTML <head> element is used as a container for metadata. It is used between <html> tag and <body> tag.

The head of an HTML document is a part whose content is not displayed in the browser. It just contains metadata about the HTML document which specifies data about the HTML document.

Metadata defines the document title, character set, styles, links, scripts, and other meta information.

Tags of Metadata in Head

HTML <meta> Tag

Description

<meta> is used to specify the character set, page description, keywords, authors and other metadata on the webpage. Metadata is mainly used by browsers, search engines, and other web services to rank your webpage better.

Description

HTML <link> Tag

Description

The <link> tag in HTML is used to link a HTML document with an external resource.

Attributes

  • rel

    the rel attribute provides with the relationship between the HTML document and the destination resource. For example rel="stylesheet" indicates that the external resource is a style sheet for the current document.

HTML <style> Tag

Description

<style> is used to style the HTML page. But it is considered to be bad to use <style> in HTML. If we want to standardize our code, we should use <link rel="stylesheet"> instead.

HTML <script> Tag

Description

HTML <script> element is used to add an external JavaScript file to current page. Actually you can write JS in the <script> instead of link a file. But to standardize your code, I suggest to link a JS file instead.
