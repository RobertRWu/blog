Memory Structure

Stack

  • Allocated from high address to low address from global perspective. But for local perspective, for example, if allocate a variable, is from low address to high address which means its lowest bit is at lower address and its highest bit in at higher address.
  • Store arguments of function calls and local variables.

Heap

  • Heap is a data structure expand from low address to high address.

  • The size of heap is confined by the virtual memory of the computer.

  • In OS, there is a free list that records all free address. When we need to allocate memory, system will traverse the list and find a qualified one. Then delete it from free list and allocate to the application.

  • System will put the redundunt memory back to the list.
  • Much slower than stack.

BSS Segment

In C, BSS segment stores global and static variables that has not been intialized.

But in C++, they all store in the same memory space.

Data Segment

In C, data segment is used to store constants and intialized global and static variables.

Text Segment

Stores code.

Allocate Memory

malloc()

Description

The memory that malloc() has allocated contains two different parts:

  • Memory to store data.
  • Memory to store additional information (cookie).

If you only use malloc() to allocate a small size of memory, then the additional part can occupy a large proportion which is really wasteful.

free()

How does free() know the size of allocated memory?

There are several ways:

  • Use a table to record the size and start address of the allocated memory.
  • Use a pointer to store the data.
  • Store additional information about the allocated memory in front of data.

new (Array New)

Implementation

  1. Inovke operator new().
  2. Use static_cast to convert the type.
  3. Invoke the constructor of object.

Properties

  • new is actually static. Because when we use new, we actually don’t have an object which means we can only call only call new from class. To achieve this, we should use static.

Reload

c++

Difference between new and malloc

  • malloc is a function but new is an operator.
  • malloc returns a void pointer but new returns a pointer with certain type.
  • If the allocation failed, malloc will return NULL but new will throw a bad_alloc.
  • We can overload new and allocate memory from other memory area.

delete

Implementation

  1. Invoke the destructor of object.
  2. Inovke operator delete().

Properties

  • If you use delete, the destructor will only be called once. But all the memory will be free. So actually memory leak is caused by the memory allocated in the object, not delete.
  • If you use delete[], the order of destructor calling is reverse (from high address object to low address object).

new()

Implementation

  1. Invoke malloc().

Usage

c++

delete()

Implementation

  1. Invoke free().

Placement New

Description

Allow us to construct objects in the memory that has already been allocated.

Usage

c++

Implementation

  1. Inovke operator new() (Another reloaded version, this function won’t allocate memory but only return the address which is specfied as argument).
  2. Use static_cast to convert the type.
  3. Invoke the constructor of object.

Properties

  • Placement new doesn’t apply for memory. So we can’t use it to construct an object whose memory hasn’t been allocated.

Reload

c++

Memory Pool

Description

When we need to new a lot of objects, it can be time-consuming and space-consuming because of cookies. So we can allocate a large space in the first and divide it into several blocks. And this is called a memory pool./

A memory pool is constituted by a free list with memory block that has already been allocated.

Implementation

  • We need to reload operator new to implement memory pool.
  • Each memory block’s size is equal to the object’s size that you are dealing with.
  • We need a pointer to the free object of the memory pool in the class.

Reload operator new

  1. If the list is empty, allocate specified memory and initialize this memory with empty objects.
  2. When we need to new an object, we just need to fetch one from the free list, and make the head of the free list point to the next one.

Reload operator delete

  1. Return back the object to the head of free list.
