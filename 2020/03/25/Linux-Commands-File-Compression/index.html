Compression

gzip

Description

Used for compressing files.

Properties

  • Can’t compress directories.
  • Won’t reserve original files.

Synopsis

shell

Options

  • -a or --ascii

    Use ASCII text mode.

  • -c or —stdout or —to-stdout 　

  • -d or --decompress or --uncompress

    Decompress a file.

  • -f or —force 　

  • -h or --help

  • -l或—list 　

  • -L或—license 　

  • -n或—no-name 　

  • -N或—name 　

  • -q或—quiet 　

  • -r或—recursive 　

  • -S<压缩字尾字符串>或——suffix

  • -t或—test 　

  • -v或—verbose 　

  • -V或—version 　

  • —best 　

  • —fast 　

Examples

shell

tar

Description

Used for backing up or archiving files.

Properties

  • Will reserve original files.

Synopsis

shell

Options

  • -c or --create

    Create a .tar file.

  • -f

    Specify the archived file’s name.

  • -j

    Use bzip2 to compress files.

  • -v

    Display details.

  • -x

    Restore files from an archived file.

  • -z

    Use gzip to compress files.

Examples
