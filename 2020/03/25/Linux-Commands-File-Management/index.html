umask

Description

Set the default mask for the new files and directories. The mask is the same conception with that in computer network. For example, if the mask is 754, then the new directories authority will be 023. It is radix-minus-one complement.

Search

find

Description

Find files.

Synopsis

shell

Options

Logic

  • -a (and)

    Find files that satisfy both requirements. Used to add filters.

    synopsis:

    find [range][pattern] -a [pattern]

  • -o (or)

    Find files that satisfy one of the requirements. Used to add filters.

Time

  • -amin (access)

    Find files which have been accessed in a certain period of time.

    pattern:

    • find [range] -amin -time

  • -cmin (change)

    Find files whose properties have been changed in a certain period of time.

  • -mmin (modify)

    Find files whose content has been modified in a certain period of time.

Name

  • -iname

    Find by name, case-insensitive.

  • -name

    Find by name. Find those files that have this name exactly, not include files that have the pattern as substring.

User

  • -group

    Find by group.

  • -user

    Find by owner.

Additional Execution

  • -exec

    Find files and execute the following command.

    synopsis:

    • find [range][pattern] -exec command {} \;

  • -ok

    The same as -exec, but will ask you if you want to execute the command.

Other

  • -inum (inode number)

    Find by the files inode number.

  • -size

    Find by size.

    synopsis:

    find [range] -size [+/-/]size

    • +n

      bigger than n blocks (A block has 512 bytes)

    • -n

      smaller than n

    • n

      equals to n

  • -type

    Find by type.

    synopsis:

    find [range] -type [f/d/l]

    • f

      file

    • d

      directory

    • l

      link

Example

shell

grep

Description

Search contents in files. This operation will display lines in the file that have the specified pattern.

Synopsis

shell

Options

  • -i (case-insensitive)

    Case-insensitive.

  • -n

    Display the row number.

  • -r or --recursive

    When the target is a directory, then search recursively.

  • -v

    Exclude lines that have the specified pattern and then display other contents of the file.

  • -E

    Use regular expression as pattern.

  • --color=auto

    Colorize searched key words.
