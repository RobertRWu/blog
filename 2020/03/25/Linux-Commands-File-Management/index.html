Display

cat

Description

  1. Display the content of a file to standard output.
  2. Create a new file.
  3. Concatenate several files

Options

  • -n

    Display the row number.

Examples

shell

tac

Description

  1. Display the content of a file upside down (from the last line to the first line) to standard output.

more

Description

Display a file by pages to standard output.

Shortcuts

  • Space or f

    Next page.

  • b or B

    Previous page.

  • Enter

    Change line.

  • q or Q

    Quit.

less

Description

Enhanced more.

Shortcuts

  • Space or f

    Next page.

  • b or B

    Previous page.

  • Enter

    Change line.

  • q or Q

    Quit.

  • h

    Help

  • /string

    Search for string forward.

  • ?string

    Search for string backward.

head

Description

Display the first several lines to standard output.

Options

  • -n

    Number of lines to display. The default value is 10.

tail

Description

Display the last several lines to standard output.

Options

  • -n

    Number of lines to display. The default value is 10.

  • -f

    Display the content dynamically.

Operation

touch

Description

Create an empty file.

Example

shell

ln (link)

Description

Create a link file.

Options

  • -s (Symbolic)

    Create a symbolic link.

Examples

shell

Synopsis

shell

Options

  • -R (Recursive)

    Change files and directories recursively.

Examples

shell

chown (change ownership)

Description

Change the ownership of a file or directory.

  • Only the root can change the ownership.

Synopsis

shell

Examples

chgrp (change group)

Description

Change the group ownership of a file or directory.

  • Only the root can change the ownership.

Synopsis

shell

Examples

shell

umask

Description

Set the default mask for the new files and directories. The mask is the same conception with that in computer network. For example, if the mask is 754, then the new directories authority will be 023. It is radix-minus-one complement.

  • The new file always does not have the access to execution.

Options

  • -S

    Use the format rwx.
