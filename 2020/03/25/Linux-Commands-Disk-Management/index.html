Information

ls (list)

Description

List all files and sub-directories of the current directory.

Options

  • -a (all)

    Print all files including hidden files.

  • -d

    Check the property of directory.

  • -h (human readable)

    With -l, print sizes in human readable format (e.g., 1K 234M 2G)

  • -i (inode)

    Print the index node number.

  • -l

    Print details of files.

  • -t

    Sort by modification time

Examples

shell

where (the order is from left to right),

  • drwxr-xr-x

    The first letter indicates the type:

    • -

      File.

    • d

      Directory.

    • l

      Symbiotic link

      rwx means reading, writing and execution access.

      2-4 letters are for the owner; 5-7 are for users in the same group; 8-10 are for others.

  • 2

    reference count of the file

  • robert

    Owner of the file.

  • robert

    Owner group of the file.

  • 4096

    The size of the file. Use byte as unit

pwd (print working directory)

Description

Show current working directory.
