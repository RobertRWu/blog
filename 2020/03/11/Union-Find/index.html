Union-find Set

Introduction

For a set $S=\{a_1, a_2, …, a_{n-1}, a_n\}$, we can get one of its partitions: $S_1,S_2,…,S_{m-1},S_m$. If we hope to determine if two elements of $S$ are in the same subset of $S$, we can use union-find set.

In a union-find set, we represent each subset with a tree. Then the union-find set is the sum of these trees and thus it is a forest.

We can use an array to represent the forest. Each node is represented with an index, and the value corresponding to the index is its root’s index.

For example:

value 0 0 0 3 3 5 6

Operation

Union

Join two trees into a single tree.

Find

This operation returns the element’s root node. If we want to determining if two elements are in the same subset, we can find their roots and check if they have the same root.

The time complexity is $O(D)$, $D$ is the depth of the node.
