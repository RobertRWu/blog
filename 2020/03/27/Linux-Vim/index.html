Shortcut Keys

Change Mode

Insert mode

  • a

    Insert after the cursor.

  • A

    Insert at the end of line.

  • i

    Insert at the cursor.

  • I

    Insert at the beginning of line.

  • o

    Insert a new line under the cursor.

  • O

    Insert a new line on the cursor.

Other

  • esc

    Back to command mode.

  • :

    Ex command mode.

Save and Quit

  • :wq

    Write and quit.

  • :w

    Write.

  • :w new_filename

    Write to new_filename.

  • :q!

    Do not write and quit.

  • :wq!

    Override, write and quit.

Display Format

  • :set nu

    Display row numbers.

  • :set nonu

    Do not display row numbers.

Locate

  • gg

    Go to the first line.

  • G

    Go to the last line.

  • nG

    Go to line n.

  • :n

    Go to line n;

  • $

    Go to the end of line.

  • ^

    Go to the beginning of line.

Delete

  • x

    Delete the character at the cursor.

  • dd

    Cut current line.

  • D

    Delete content between the cursor and end of current line.

  • dG

    Delete content between the cursor and end of current file.

  • :n1,n2d

    Delete lines between n1 and n2.

Copy and Cut

  • yy

    Copy current line.

  • nyy

    Copy n lines below current line.

  • dd

    Cut current line.

  • ndd

    Cut n lines below current line.

  • p

    Paste below current line.

  • P

    Paste above current line.

Replace Character and Undo

  • r

    Replace the character after the cursor.

  • R

    Enter replace mode.

  • u

    Undo the last change.

Search and Replace

  • /string

    Search for string.

  • n

    Next searching target.

  • :%s/old content/new content/g or c

    Replace string in the whole file.

    g means no asking for check.

    c means asking for check.

  • n1,n2s/old content/new content/g or c

    Replace string between line n1 and n2.

Import

  • :r file_path

    Import content from files below the cursor.

Other

  • !command

    Execute command without quitting vim.
