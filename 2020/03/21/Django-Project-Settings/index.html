Database

python

  • ENGINE

    The database engine that you want to use. Django supports following databases:

    • MySQL

      django.db.backends.mysql

    • PostGreSQL

      django.db.backends.postgresql_psycopg2

    • Oracle

      django.db.backends.oracle

    • MongoDB

      django.db.backends.mongodb_engine

    • SQLite

      django.db.backends.sqlite3

  • NAME

    The name of the database that you created, includes the path and name.
