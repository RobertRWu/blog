Linux Configuration Files
Created 2020-03-29| Updated 2020-03-31|Computer ScienceLinux
Word count:136|Reading time: 1 min
| Post View:|comments:

Directories

Root Directory

Path Description
/bin Stores commands.
/boot Stores core files for system booting.
/dev Stores devices.
/etc Store all configuration files for system management.
/home Home directory for users.
/lib Stores library of C compiler and part of compilers.
/media
/mnt Used for user to mount device.
/opt Stores third party applications.
/proc A kind of pseudo file system, stores infomation about hardware and process.
/usr Stores many user’s applications and files.

Configuration Files

/etc

Path Description
/etc/resolv.conf DNS configuration file.
/etc/hosts Domain name resolution.
/etc/gateways Set routes.
/etc/fstab File system mount table.
/etc/rc.local
/etc/profile Set environment variables.
/etc/bashrc Set aliases.

/proc

Path Description
/proc/cpuinfo Hardware Information about CPU.
/proc/swaps Information about the swap.
