CSS Contextual Selector

Introduction

Contextual selector uses document’s contextual relationship to determine tags’ style. There are three different kinds of contextual selector:

  • Descendant Selector
  • Child Selector
  • Adjacent Sibling Selector

Descendant Selector

Descendant selector will choose every descendant elements, including child, grandchild, etc. .
