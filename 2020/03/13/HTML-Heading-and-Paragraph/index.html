Description

HTML Paragraph

Description

HTML uses <p> to define a paragraph. One thing you should know is that continuous spaces in the paragraph will be replaced with only one space. And if you want to start a new line, you should use tag <br /> instead of enter. Besides, if you want to add a horizontal line to separate two sections, you can use tag <hr />.
